IoT medical devices in healthcare clinics are increasingly vulnerable to cyberattacks.
Existing systems require manual MUD profile creation with no explainability for detected threats.
This project proposes an automated security framework using Auto MUD profiles and SHAP explainability.
ECU-IoHT dataset is used with K-Means clustering to automatically generate MUD profiles.
Seven ML models are trained and compared — RF, XGBoost, CatBoost, MLP, SVM, KNN and Logistic Regression.
SHAP explainability identifies and explains the reason behind every detected threat.
XGBoost achieved 99.4% accuracy and MLP showed highest improvement of 2.80% with Auto MUD.
A live dashboard is built for real time monitoring of all hospital IoT devices.
The proposed framework makes IoT security automatic, explainable and practically deployable in real hospitals.
